Tiny uppercase, wide-tracked badge for statuses and categories — `WATCHED`, `WATCHLIST`, `TV`, `PRIORITY`, `LIVE NOW`. Always short; rendered at 10px bold.

```jsx
<Badge tone="emerald" icon="check-circle-2">Watched</Badge>
<Badge tone="violet" icon="bookmark">Watchlist</Badge>
<Badge tone="rose" dot>Live Now</Badge>
<Badge tone="neutral">TV</Badge>
```

Tones: `violet` / `orange` / `rose` / `emerald` / `amber` / `neutral`. `dot` adds a pulsing status dot. Icons are Lucide names.
