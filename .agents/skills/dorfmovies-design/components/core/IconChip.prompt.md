Round accent-tinted icon well — the badge at the top-left of bento stat cards and inside the "now watching" card.

```jsx
<IconChip icon="calendar" tone="orange" />
<IconChip icon="flame" tone="violet" />
<IconChip icon="play" tone="rose" size={48} glow />
```

`tone`: `violet` / `orange` / `rose` / `emerald`. Pass `hover` (e.g. tied to the parent card's hover state) to trigger the 110% scale. Icons are Lucide names.
