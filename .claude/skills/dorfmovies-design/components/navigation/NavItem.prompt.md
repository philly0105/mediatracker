Sidebar nav row used in the fixed 256px rail.

```jsx
<NavItem icon="home" label="Dashboard" active />
<NavItem icon="search" label="Search" />
<NavItem icon="bar-chart-3" label="Stats" />
```

Active row gets a glass pill + violet icon. Stack them in a column with ~4px gaps. Icons are Lucide names (`home, search, calendar, film, tv, list-todo, library, sparkles, swords, layers, bar-chart-3, upload, settings`).
