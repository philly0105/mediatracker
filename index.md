# DorfMovies

## Summary
A personal web app for tracking movies and TV shows — logging what you've watched, rating them, and managing a prioritized watchlist.

## Goals
- Track watched movies and TV shows with ratings
- Maintain a watchlist with priority levels (how badly you want to watch something)
- Browse, search, and filter your media history
- Pull metadata (posters, descriptions, cast) from an external API
